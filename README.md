## Basic Usage

克隆本仓库
```Bash
git clone https://github.com/Sricor/yiban.git
```

安装依赖
```Bash
cd yiban
pip install -r requirements.txt
```

修改 index.py 中 `config` 配置

```Bash
python3 index.py
```
